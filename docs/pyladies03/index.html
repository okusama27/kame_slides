テーマ選び¶

  • 新規追加された機能を探す
  • プログラマが混乱しやすい機能（同じような機能だけど、複数ある）
  • 他の言語で流行っている本、話題をPythonに置き換えて考える
  • 毎年、同じテーマを出し続ける（徐々に聞きたいファンが増えてくる）
  • まだ誰も手を付けていないこれから流行りそうなテーマを探す
